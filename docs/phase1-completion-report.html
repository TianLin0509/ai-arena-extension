④ E2E 验证清单（需你手动跑） 用户介入

按 CLAUDE.md 测试铁律，UI / IPC 流程必须真实加载扩展验证。在 Chrome 加载 src/ 目录后，逐项打勾：

  • 1. chrome://extensions → 开启开发者模式 → 加载已解压扩展 → 选 C:\Users\lintian\AI_debate\ai-arena-extension\src
  • 2. 打开 sidepanel → 顶部出现"🪟 群聊"蓝色按钮，版本徽章显示 v4.0.0
  • 3. 加 Claude + Gemini + ChatGPT 三个参与者（用并列模式） → 3 个独立 chrome window 弹出
  • 4. 点 sidepanel "🪟 群聊" → popup 窗口在主屏右半弹出（800×900）
  • 5. popup 输入 hello + Ctrl+Enter → 3 个 AI 气泡显示 typing → 1.5s 后开始填充内容 → 流完显示完整回答
  • 6. popup 输入 @Claude 你好 + 看候选菜单 → 选 Claude → 输入框变 @Claude → 再敲 你好 + Ctrl+Enter → 只有 Claude 气泡触发，Gemini/ChatGPT 不响应
  • 7. 让 Claude 写 artifact：发 写个 React todo list 用 artifact → popup Claude 气泡末尾出现 📦 含 artifact ↗ 在 Claude 查看 pill
  • 8. 点 pill → 自动切到 Claude window/tab
  • 9. 关闭 popup → 点 sidepanel "🪟 群聊"重开 → 之前的对话气泡全部恢复
  • 10. 拖动 popup 到不同位置 → 关闭 → 重开 → popup 位置还原
  • 11. popup 顶部点 🗑 → 确认清空 → 消息列空 → AI 原页不受影响
  • 12. 关闭其中一个 AI 标签页 → popup 再发消息 → 该 AI 气泡显示 ⚠ XX 已断开，其他 AI 正常
