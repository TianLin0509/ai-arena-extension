🚀 安装教程 · 小白版 3 分钟搞定
从 GitHub 下载到第一次广播，5 步完成
1

从 GitHub 下载扩展包

  1. 打开 GitHub 仓库：github.com/TianLin0509/ai-arena-extension
  2. 点击页面右上角的绿色 <> Code 按钮
  3. 在弹出菜单底部点 Download ZIP
  4. 浏览器自动下载 ai-arena-extension-main.zip 文件

💡 没装过 Git？没关系，Download ZIP 不需要任何工具。

⚠️ 公司网络访问 GitHub 可能慢，建议挂代理或用手机热点下完再拷过来。

🖼️ 此处插入截图：GitHub Download ZIP 按钮位置（建议自补 github-download.png）
2

解压 zip

  1. 找到下载好的 zip（默认在 下载 文件夹）
  2. 右键 → "全部解压" → 选个能记住的目录（建议桌面）
  3. 解压完会得到一个文件夹 ai-arena-extension-main
3

在 Chrome 中加载扩展

  1. 打开 Chrome 浏览器
  2. 在地址栏粘贴：chrome://extensions → 回车
  3. 打开右上角的 "开发者模式" 开关
  4. 点击左上角 "加载已解压的扩展程序" 按钮
  5. 在弹出的文件夹选择框里：选刚才解压目录里的 dist/github 子文件夹
  6. 看到一张 "AI Arena" 卡片出现 ✓ = 安装成功

⚠️ 重要：要选 dist/github 子文件夹，不是根目录。如果选错会报错"找不到 manifest.json"。

🖼️ 此处插入截图：Chrome 扩展页加载成功（建议自补 chrome-extensions-loaded.png）

🚨 常见问题 FAQ

为什么 AI 不回答？
① 检查是否登录该 AI 网站（在 Chrome 里手动打开它确认能正常用）；② 看 chrome://extensions AI Arena 卡片的错误日志；③ 该 AI 最近改了网页 DOM —— v4.8 有 4 层兜底，极少数情况仍可能漏，告诉作者马上修。
数据安全吗？
100% 本地。扩展不收集、不上传任何对话内容，只在你的浏览器和已登录的 AI 官网之间流动。源码全部开源在 GitHub，可自查。
公司内网能用吗？
国内 5 家 AI（DeepSeek / 豆包 / 通义 / Kimi / 元宝）在公司网络下应该可用。海外 4 家（Claude / Gemini / ChatGPT / Grok）需要解决网络可达性。
要收费吗？
永久免费，MIT 开源协议。
怎么提反馈？
GitHub Issues / Beta 群（见下方）/ 回复本帖留言。
